# Fake Product Review Monitoring System

A web application that detects fake product reviews using sentiment analysis and machine learning techniques.

## Tech Stack
- Node.js
- HTML, CSS, JavaScript

## Features
- Review classification (fake/genuine)
- Sentiment analysis
- Simple UI for monitoring reviews
